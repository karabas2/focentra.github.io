# Focentra | Anti-Distraction Smart Glasses

Focentra is a premium landing page for a futuristic pair of smart glasses designed to physically eliminate distractions and amplify focus using active vision-shielding technology.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
